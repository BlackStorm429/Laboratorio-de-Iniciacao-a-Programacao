App Inventor

Projeto criado no App Inventor para a disciplina de Laboratório de Iniciação à Programação. O trabalho é um aplicativo feito para o sistema Android, mas que também funciona no sistema IOS. É um app simples que permite o usuário digitar algo para que o aplicativo fale e o app também disponibiliza um vídeo para o usuário assistir.
